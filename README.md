# AI Personal Assistant

An AI-powered Personal Assistant that can answer user queries and summarize emails quickly and efficiently through a simple web interface.

## Features

* 🤖 AI-powered conversational assistant
* 📧 Email summarization
* ❓ Natural language query answering
* ⚡ Fast and responsive web interface
* 🎨 Clean and user-friendly design
* 🔒 Secure API key handling using environment variables

## Tech Stack

* Python
* Flask
* HTML/CSS
* OpenAI API

## Installation

1. Clone the repository

```bash
git clone https://github.com/niteash-patel/AI-Personal-Assistant.git
cd AI-Personal-Assistant
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a `.env` file

```env
OPENAI_API_KEY=your_api_key_here
```

4. Run the application

```bash
python main.py
```

## Usage

* Enter a question to get an AI-generated response.
* Paste an email to receive a concise summary.
* Use the assistant for productivity and information retrieval tasks.
